# weather_app
A simple and userfriendly  weather application built using javascript that shows real-time weather information of any city using the open weather API

 #Features :
-> search weather by city name.
-> Display temperature in celsius 
-> shows weather condition (clear.cloudy , rain ,etc .)
-> displays humidity and windspeed 
-> fast and responsive design 

#Technologies used :
HTML
CSS
Javascript
Openweather API
